# Events

The `epochs` module emits the following events:

## BeginBlocker

  Type           Attribute Key   Attribute Value
  --------------; ---------------; -----------------;
  epoch\_start   epoch\_number   {epoch\_number}
  epoch\_start   start\_time     {start\_time}

## EndBlocker

  Type         Attribute Key   Attribute Value
  ------------; ---------------; -----------------;
  epoch\_end   epoch\_number   {epoch\_number}
